üß™ Markdown + HTML = Weapon

At first glance, the Markdown parser seems limited‚Ä¶ until we test raw HTML.

<iframe src="http://localhost:5000/admin"></iframe>

The PDF download includes the contents of the iframe. Jackpot.

Looks like the PDF renderer is executing the HTML inside the Markdown on the server, and the iframe resolves to localhost, allowing access to the restricted admin panel.

ü™ù Flag Retrieval (and Why This Works)

Inside the PDF (zoomed in, of course), we can now view the /admin page's content rendered neatly ‚Äî including the flag.

This works because wkhtmltopdf, a common tool for HTML-to-PDF conversion, is used behind the scenes. When the server renders our Markdown (with embedded HTML), it follows the link to localhost:5000/admin, bypassing the 403 restriction we hit from the browser.
